# Instrukcja

Umieść swoje CV w tym folderze jako plik PDF o nazwie:
`Mateusz_Siuba_CV.pdf`

Wtedy przycisk "Pobierz CV" w nawigacji będzie działał poprawnie.

## Jak przygotować CV:
1. Stwórz CV w Word/Google Docs/Canva
2. Eksportuj jako PDF
3. Zmień nazwę pliku na: `Mateusz_Siuba_CV.pdf`
4. Umieść tutaj (w folderze public/cv/)
